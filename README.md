## TP 1: Juego 2D

# Consigna:
Crear un videojuego 2D con Godot Engine y utilizando GDScript.
El juego deberá tener más de una pantalla, por ejemplo:
- Splash Screen
- Menu principal
- Créditos
- Pantalla del juego en sí
- etc…

Tiene que tener texturas, fuentes y sonidos. Éstos recursos pueden ser descargados de internet o hechos por uno mismo.
Además deberá contar con una condición de victoria y otra de derrota.
Se deberá subir el código del juego a un repositorio de github.
Se deberá exportar el juego en formato HTML o .exe y subirlo a la página de itch.io.
Está página deberá tener información del juego con screenshots y una breve descripción del mismo.

# Ejemplos:
- https://stranker.itch.io/rhum-and-pirates
- https://strangescaffold.itch.io/i-am-your-beast
- https://anuke.itch.io/mindustry

# Se va a evaluar:
- Diseño del juego
- Organización del proyecto
- Organización de las escenas
- El uso de señales o call groups
- El uso de las layers de colisión 2D
- El uso de Autoloads
- El uso de partículas y sistema de animación

# Formato de entrega:
Se debe subir un .txt al aula virtual con dos links: uno a la página de itch.io de su juego y el otro para el github de su proyecto.
Opcionalmente, se puede crear el repositorio para el juego a partir de este template: https://github.com/image-campus-2025-trayecto-godot/base-para-tp que automatiza la exportación del juego a itch.io.
